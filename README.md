# FlashCards Privacy Policy Web App

A premium, responsive, and modern Privacy Policy web application built for the **FlashCards** mobile app. This site provides users with clear information about data collection, storage, and security practices while maintaining a high-end visual aesthetic.

## ✨ Features

- **Premium Design**: Modern UI with glassmorphism, smooth gradients, and vibrant accents.
- **Interactive Navigation**: Sticky table of contents with active section tracking.
- **Mobile Optimized**: Fully responsive layout with a sleek mobile menu.
- **Deep Link Integration**: Includes "Open App" functionality with smart fallback and QR code generation for easy mobile access.
- **Micro-animations**: Subtle hover effects and transitions using CSS and Lucide React icons.
- **SEO Ready**: Optimized meta tags and semantic HTML structure.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **QR Codes**: [qrcode.react](https://www.npmjs.com/package/qrcode.react)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create a production-ready bundle:

```bash
npm run build
```

The output will be in the `dist` folder, ready for deployment on platforms like Netlify, Vercel, or GitHub Pages.

## 📂 Project Structure

- `src/PrivacyPolicy.tsx`: The main application component containing the layout and logic.
- `public/`: Static assets including the logo and background patterns.
- `tailwind.config.js`: Custom theme configuration for the brand's color palette.
